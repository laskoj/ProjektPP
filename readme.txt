Kompilacja pliku odbywa się poprzez użycie oprogramowania IntellJ IDEA Community Edition, w moim przypadku wersja to 2024.3.1.1
Build #IC-243.22562.218, built on December 18, 2024.
* Uruchamiamy plik Main.java poprzez użycie ikony na górnej belce (SHIFT + F10)

Uruchamia się nam okno w możliwościa wpisania danych 
-Student ID
-Name
-Age
-Grade
Po wpisaniu poprawnych danych np: 50015, Fojut, 24, 5. Program zwróci nam informacje "Student added successfully!"
Możemy zastopować nasz program SMS poprzez użycie czerwonego kwadratu na górnej belce (CTRL + F2)

Aby uruchomić nasza baze danych students.db, potrzebujemy składnika sqlite-jdbc-3.47.2.0.jar. Dodajemy go poprzez naciśniecie CTRL + ALT + S i wchodzimy w Plugins, a następnie szukamy SimpleSqliteBrowser i instalujemy go. 
Od teraz możemy uruchomic naszą baze poprzez dwuklik z listy plików (students.db) w programie IntellJ.
W tym oknie możemy odświeżyć nasza baze i zweryfikować obecność wyżej podanego studenta. 

Dodatkowo możemy zainstalować program DB Browser (SQLite) i również tam prowadzić naszą baze danych.
